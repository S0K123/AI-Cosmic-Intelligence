## 📥 Installation
Clone this repo, create and activate the venv, then:

```bash
pip install -r requirements.txt
```

## ▶️ Run the App
Use the virtual environment:
```bash
streamlit run app_final.py
```
(Then visit http://localhost:8503

## 📊 App Sections
- **Home** - Quick stats and project overview!
- **About Project** - AI models, data sources, tech stack!
- **Dataset Explorer** - Search, filter, and visualize!
- **AI Prediction** - AI ensemble predictions!
- **Planet Ranking** - Full ranking!
- **3D Spatial Explorer** - Interactive 3D orbital visualization!
- **Simulation** - Simulate orbital dynamics!
- **Research Analytics** - Plots and analysis!
- **Model Comparison** - Compare all models!
- **Settings** - Config options!
- **Export Results** - Download everything!
